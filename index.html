TYPES OF CSS STYLINGS

Internal CSS is a method of adding CSS (Cascading Style Sheets) directly within an HTML document. It is used to apply specific styles to a particular HTML page. The CSS rules are written inside a style element, which is placed within the head section of the HTML file.

Inline CSS is a method of applying styles directly to individual HTML elements using the style attribute within the element's opening tag. This method is useful for applying unique styling to a single element without affecting other elements on the page.

External CSS is a method of applying CSS (Cascading Style Sheets) by linking to an external .css file, which contains all the styling rules. This method is efficient for styling multiple HTML pages with a consistent look and feel, as changes made to the external CSS file automatically apply to all linked pages.
